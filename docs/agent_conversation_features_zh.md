# 智能体对话与评估 — 功能描述

## 1. 可自定义的家长提示词

目前系统中教练（coach）的提示词已经支持自定义。本功能将同样的灵活性扩展到家长侧——用户可以直接提供自定义的家长提示词，而不必依赖预设的家长档案。

使用自定义提示词时，系统会在输出中标记 `"parent_profile": "custom"`，便于区分。

## 2. 生成与评估不设置超时

对话生成（多轮 LLM 交互）和 3 维度评估可能需要 2 到 10 分钟以上。系统需确保长时间运行的任务不会因超时而被中断，无论是在后端还是前端层面。

## 3. 异步任务体验

生成过程本质上是耗时的。用户体验应将其视为异步任务，而非同步的请求-响应：

- 用户点击"生成"后，立即获得确认，UI 显示任务卡片
- 用户可以离开当前页面，任务卡片在侧边栏 / 任务列表中持久存在
- 任务卡片实时更新进度："第 2/5 轮对话..."、"正在评估 TES..."、"完成 ✓"
- 完成后，用户点击查看对话和评估结果
- 不会有加载动画阻塞整个界面——生成是一个后台作业
