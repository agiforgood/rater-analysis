# 下一轮专家评分的改进建议

**日期：** 2026-03-07
**背景：** 基于首轮36人评分研究的分析结果，规划新一轮专家评分数据采集。

---

## 采集前需要做的改动（按优先级排列）

### 1. 校准会议（必须做）

正式评分前，所有专家评分者先对同样的2–3段对话独立评分，然后当面讨论分歧。仅这一步就可能将评分者方差（目前占总方差的54%）减少一半。

### 2. 立即修改反向计分题

将WAI的item4和item10改写为正向表述。这是一个5分钟就能完成的修改，可以消除已知的混淆来源。没有理由在一个已知的设计缺陷上继续收集带噪声的数据。

- **item4** 原文："双方对试图达成的治疗目标，存在疑虑或缺乏理解" → 改为正向："咨访双方对治疗目标有清晰的共识和理解"
- **item10** 原文："来访者和咨询师对来访者的真实问题是什么有不同的看法" → 改为正向："来访者和咨询师对来访者的真实问题有一致的看法"

### 3. 在评分标准中添加锚定示例

为每个题目提供1–2段简短的对话摘录，展示"2分"、"4分"和"6分"分别是什么样的。优先处理分歧最大的题目：温暖（q4）、接纳感受（q8）、情感共鸣（q3）、内在同频（q5）。

### 4. 决定TES量表的去留

TES在本次研究中的ICC为0.003–0.057——基本等于零。可选方案：

| 方案 | 优点 | 缺点 |
|------|------|------|
| **保留TES，添加锚定示例** | 保持可比性；可能解决问题 | 如果锚定示例不够，浪费专家时间 |
| **精简TES至3–4题** | 减轻认知负担；可能改善区分度 | 失去细粒度 |
| **完全去掉TES** | 集中资源做有效的部分（WAI） | 失去共情测量 |
| **先做试点（推荐）** | 成本低；先验证再投入 | 略微推迟正式采集 |

**建议：** 先做试点。让3–5位专家对3段对话用修订版TES评分。如果ICC提升到0.3以上，继续推进；如果没有，就砍掉或大幅简化TES。

### 5. 将无条件积极关注扩展为3–4个题目

目前只有1个题目——无法计算ICC，无法检测作答模式。建议增加子维度：
- 非评判接纳：咨询师对来访者的感受和行为展现出非评判的态度
- 自主性支持：咨询师尊重来访者的自主性，而非试图控制或引导
- 真诚关怀：咨询师表现出对来访者作为个体的真诚兴趣和关心

### 6. 设计决策：评原来的对话还是新对话？

| 方案 | 优点 | 缺点 |
|------|------|------|
| 评相同的20段对话 | 可直接比较专家与非专家的一致性；可验证贝叶斯估计 | 专家可能觉得重复 |
| 评全新对话 | 更广的覆盖面 | 失去可比性 |
| **重叠设计（推荐）** | 评原来20段中的10段 + 10段新对话 | 兼顾可比性与覆盖面 |

## 试点方案

### 目标
验证修订后的工具能否产生可用的评分者间一致性（ICC > 0.3），然后再决定是否进行正式的专家采集。

### 参与者
3–5位专家评分者

### 材料
- 3段对话，覆盖质量高中低：一段高质量（#10）、一段中等（#8）、一段低质量（#14）
- 修订版评分标准：
  - 改写后的item4和item10
  - TES题目的锚定示例
  - 扩展后的无条件积极关注量表（3–4题）

### 流程

1. **校准环节（30分钟）**
   - 所有专家独立评分对话#10
   - 集体讨论：对比评分，讨论分歧
   - 统一对评分标准的理解
   - 注意：不要重新评分——目的是达成共识，不是强迫一致

2. **独立评分（45分钟）**
   - 每位专家独立评分对话#8和#14
   - 三个量表全部评：TES（修订版）、WAI（item4/10改写后）、无条件积极关注（扩展版）

3. **分析**
   - 按维度计算ICC(2,1)
   - 与原始研究的基线对比
   - 决策标准：
     - ICC > 0.3 → 推进正式采集
     - ICC 0.15–0.3 → 进一步修改评分标准，重新试点
     - ICC < 0.15 → 砍掉该量表或从根本上重新设计

### 时间线
- 第1天：准备修订版评分标准和锚定示例
- 第2天：校准会议 + 独立评分
- 第3天：分析试点结果，做出是否推进的决定
- 第4天起：正式专家采集（如果决定推进）

## 修订版评分标准模板

### WAI题目——改写

仅改写反向计分题：

**Item 4（原为反向题）：**
- 原文："双方对试图达成的治疗目标，存在疑虑或缺乏理解"
- 新文："咨访双方对治疗目标有清晰的共识和理解"
- 量尺：1（完全不符合）— 7（完全符合）

**Item 10（原为反向题）：**
- 原文："来访者和咨询师对来访者的真实问题是什么有不同的看法"
- 新文："来访者和咨询师对来访者的真实问题有一致的看法"
- 量尺：1（完全不符合）— 7（完全符合）

### TES题目——锚定示例格式

每个TES题目按以下格式添加锚定示例：

> **温暖 Warmth（q4）**
>
> 咨询师的温暖体现在友好、亲切、真诚的说话方式，支持来访者的自我表达。
>
> **2分示例：**
> [插入一段2–3轮对话摘录，展示温暖感很低的情况——机械化的、疏离的回应]
>
> **4分示例：**
> [插入一段2–3轮对话摘录，展示中等温暖——礼貌、支持性但不够深入个人化]
>
> **6分示例：**
> [插入一段2–3轮对话摘录，展示高温暖——真诚关怀的语气、个人化的投入、肯定性的语言]

*（锚定示例应从现有的20段对话中选取——选择清晰、无歧义的、校准评分者达成一致的片段。）*

### 无条件积极关注——扩展后的题目

原来1个题目 → 扩展为4个题目：

1. **非评判接纳：** 咨询师对来访者的感受和行为展现出非评判的态度（1–5）
2. **自主性支持：** 咨询师尊重来访者的自主性，而非试图控制或引导（1–5）
3. **真诚关怀：** 咨询师表现出对来访者作为个体的真诚兴趣和关心（1–5）
4. **整体积极关注等级：** [保留原有5阶段评分]（1–5）

## 正式采集的通过标准

试点结束后，满足以下条件才推进正式采集：

- [ ] WAI ICC(2,1) > 0.3（原始研究为0.07–0.10）
- [ ] TES ICC(2,1) > 0.15（原始研究为0.02–0.05）
- [ ] 无条件积极关注 ICC(2,1) 可计算且 > 0.2
- [ ] 改写后的题目无双峰分布
- [ ] 专家反馈确认评分标准清晰可用
