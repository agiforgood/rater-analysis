# 当前评分数据的诚实评估与应对方案

**日期：** 2026-03-07
**数据集：** 36名评分者 × 20段对话 × 3个维度（TES共情、无条件积极关注、WAI咨访同盟）

---

## 一、现状评估：评分数据质量如何？

### 核心发现：分数里大部分是"噪声"，不是"信号"

通过多面Rasch模型（MFRM）的方差分解，我们发现评分中的变异来源如下：

| 变异来源 | 占比 | 含义 |
|----------|------|------|
| **评分者严厉度** | **54%** | 分数高低主要取决于"谁在评" |
| **题目难度** | **38%** | 其次取决于"评的是哪道题" |
| **对话质量** | **8%** | 我们真正想测量的东西，只占8% |

也就是说，一段对话得到高分还是低分，**主要不是因为这段对话质量好或差**，而是因为碰上了一个手松的评分者，或者碰上了一道容易给高分的题目。

### 具体有多严重？

**评分者之间的差异：**
- 最宽松的评分者：平均分 6.19（满分7分，几乎什么都给高分）
- 最严格的评分者：平均分 3.27（几乎什么都给低分）
- 差距：在7分制量表上相差近 **3分**
- 有一位评分者对所有20段对话都给了 **完全相同的4分**，完全没有区分

**各量表的一致性（ICC）：**

ICC是衡量评分者间一致性的指标，范围0–1，越高越好。

| 量表 | ICC(2,1) 单个评分者 | 解读 | ICC(2,36) 36人平均 | 解读 |
|------|---------------------|------|---------------------|------|
| WAI 咨访同盟 | 0.07–0.10 | 极差 | **0.73–0.80** | 良好 |
| 无条件积极关注 | 0.06 | 极差 | **0.70** | 中等 |
| TES 共情 | 0.02–0.05 | 接近零 | **0.42–0.67** | 差–中等 |

**单个评分者的评分几乎不可信**——等同于随机噪声。但36人的平均值是可用的，至少对WAI量表如此。

### 哪些题目分歧最大？

分歧最大的两道题恰好是WAI中的**反向计分题**（item4、item10），比正常题目高出0.26个标准差。部分评分者理解为"分越高=疑虑越多"，另一部分人在心里做了反转。这是一个设计缺陷。

将所有题目按分歧程度分为两组：

**分歧较小（容易达成一致）：** 关切、回应性、信任、目标一致、专业信心——这些是**可观察的行为**

**分歧较大（难以达成一致）：** 温暖、接纳感受、情感共鸣、内在同频——这些需要**主观的情感推断**

### 分数分布有没有"两极分化"的情况？

我们检查了所有440个（对话×题目）组合的分布形态。结果：

- **98.2%** 是单峰分布（评分者大致集中在一个区间）——取平均值是合理的
- **1.8%**（8个组合）出现双峰分布（评分者分成两个阵营）
- 双峰情况主要集中在**对话24**（本身有争议的对话）和**反向计分题**

## 二、我们怎么处理这些数据？

既然单个评分者不可信，我们用了三种统计方法从噪声中提取"主体间共识"（intersubjective quality）。

### 方法1：评分者中心化平均

**原理：** 每个评分者有自己的"基线"（有人习惯给高分，有人习惯给低分）。先减去每个人的个人平均分，再加上总体平均分，这样就消除了基线差异，只保留每个评分者对不同对话的相对排序。

```
调整后分数 = 原始分数 - 该评分者平均分 + 总体平均分
```

**结果：** 调整前后的对话排名几乎不变（最大变动仅1位）。这是因为本研究采用了**全交叉设计**（所有评分者评所有对话），严格和宽松的评分者均匀分布，偏差在简单平均中就已经自然抵消了。

**结论：** 原始平均值本身已经是稳健的。中心化验证了这一点，但没有改变结果。

### 方法2：多面Rasch模型（MFRM）

**原理：** 这是专门为"多个评分者、多个题目、多个被评对象"设计的模型。它同时估计三个参数：

- **对话质量 θ：** 每段对话的"公平分数"，已剔除评分者和题目效应
- **评分者严厉度 δ：** 每个评分者有多严格/宽松
- **题目难度 β：** 每道题有多难得到高分

模型公式：`分数 = μ + θ(对话) - δ(评分者) - β(题目)`

**结果：**

对话质量排名（公平分数）：

| 排名 | 对话编号 | 公平分数 |
|------|----------|----------|
| 1 | 10 | 5.11 |
| 2 | 2 | 4.94 |
| 3 | 3 | 4.91 |
| ... | | |
| 19 | 18 | 4.14 |
| 20 | 14 | 4.12 |

评分者严厉度：

| 评分者 | 严厉度 δ | 含义 |
|--------|----------|------|
| 549 | +1.33 | 最严格 |
| 189 | -1.59 | 最宽松 |
| 差距 | 2.92 | 严格与宽松的总跨度 |

最难的题目：item4（WAI反向题，β = +1.49）；最容易的题目：q9（TES回应性，β = -0.60）

### 方法3：层次贝叶斯模型

**原理：** 与MFRM类似，但额外提供**不确定性区间**。它不仅告诉你"对话10的质量是5.20"，还告诉你"我们有95%的把握，真实质量在5.14到5.27之间"。

模型结构：
```
分数 = μ + 对话质量(随机效应) + 评分者偏差(随机效应) + 题目效应(固定效应) + 残差
```

**结果：**

| 排名 | 对话编号 | 质量估计 | 95%可信区间 | 置信度 |
|------|----------|----------|-------------|--------|
| 1 | 10 | 5.20 | [5.14, 5.27] | 高 |
| 2 | 2 | 5.04 | [4.98, 5.10] | 高 |
| 3 | 3 | 5.02 | [4.96, 5.08] | 高 |
| 4 | 4 | 4.82 | — | 高 |
| 5 | 1 | 4.77 | — | 高 |
| ... | | | | |
| 19 | 18 | 4.31 | [4.25, 4.37] | 高 |
| 20 | 14 | 4.29 | [4.23, 4.35] | 高 |

可信区间宽度约0.13分。在190对对话组合中，**141对可以显著区分**（区间不重叠）。

方差分解：

| 来源 | 方差 | 占比 |
|------|------|------|
| 评分者偏差 | 0.40 | 24.7% |
| 对话质量 | 0.41 | 25.1% |
| 残差噪声 | 0.81 | 50.2% |

### 三种方法的对比

| 方面 | 中心化平均 | MFRM | 贝叶斯模型 |
|------|-----------|------|-----------|
| 排名结果 | 与原始平均几乎一致 | 几乎一致 | 几乎一致 |
| 消除评分者偏差 | ✓ | ✓ | ✓ |
| 消除题目效应 | ✗ | ✓ | ✓ |
| 提供不确定性 | ✗ | ✗ | ✓ |
| 复杂度 | 低 | 中 | 高 |

**三种方法殊途同归：对话排名高度一致。** 这归功于全交叉设计。贝叶斯模型的独特价值在于提供了可信区间——当两段对话的区间重叠时，应将它们视为质量相当，而不是硬要分出高下。

## 三、结论

### 好消息

1. **对话排名是可信的。** 尽管单个评分者的分数噪声很大，36人平均后的排名在所有统计方法下都是稳定的。对话10质量最高、对话14质量最低，这个结论是稳健的。

2. **全交叉设计拯救了这组数据。** 因为所有评分者评了所有对话，偏差自然抵消。如果当初用的是不同评分者评不同对话，这组数据就无法使用。

3. **WAI量表在36人平均后达到了良好的一致性**（ICC(2,36) = 0.73–0.80），可以作为对话质量的主要依据。

### 坏消息

1. **TES共情量表在文本评分场景下基本失效。** ICC接近零，评分者对同一段对话的共情评价天差地别。这不是评分者的问题，而是量表本身不适合纯文本评分。

2. **单个评分者的评分不可用。** 如果只有1–2个评分者，得到的分数几乎等于随机数。至少需要10–15个评分者才能获得勉强可用的一致性。

3. **反向计分题制造了不必要的混乱。** WAI的item4和item10是分歧最大的两道题，且出现了双峰分布——评分者在"这道题该怎么理解"上就已经分裂了。

### 使用建议

- **推荐使用贝叶斯模型的质量估计值**作为每段对话的"主体间质量"得分
- **当两段对话的95%可信区间重叠时，视为质量相当**
- **WAI分数可信度最高**，应作为对话质量评估的主要依据
- **TES分数仅供参考**，不应作为重要决策的依据
- **无条件积极关注只有1道题**，信息量有限，建议下轮扩展
